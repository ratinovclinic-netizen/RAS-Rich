# R.I.C.H. — инвестиционный калькулятор

Интерактивный калькулятор **Ratinov Invest Club of Health** для подготовки персонального инвестиционного предложения.

## Возможности

- расчёт доходного капитала в сомах;
- фиксированная ставка по сумме, сроку и периоду выплат;
- капитализация процентов и сравнение сценариев;
- инвестиция в долю клиники и акции RS Holding;
- персональная цель инвестора;
- график начислений;
- печатный оффер и заявление о намерениях;
- адаптивная версия для компьютера и телефона.

## Публичная версия

https://rich-invest-club.andrewratinov.chatgpt.site/

## Локальный запуск

```bash
pnpm install
pnpm run dev
```

Проверка проекта:

```bash
pnpm run build
node --test tests/rendered-html.test.mjs
```

## Работа с данными

Не добавляйте в репозиторий персональные данные инвесторов, подписанные договоры или ключи доступа.
